### 📁 Структура файлов

```
purchases/
├── templates/
│   ├── base.html              # Базовый шаблон со стилями
│   └── purchases/
│       ├── request_form.html  # Форма создания заявки
│       └── request_list.html  # Список заявок
└── static/
    └── css/
        └── style.css          # Кастомные стили и анимации
```

### 🎯 Ключевые особенности

1. **Bootstrap 5** - современный и надежный CSS-фреймворк
2. **CSS Gradients** - плавные переходы цветов
3. **Flexbox & Grid** - современные подходы к макету
4. **CSS Variables** - легко менять цвета и значения
5. **Media Queries** - оптимизация для всех размеров экранов

### 🚀 Как использовать

1. Дизайн полностью интегрирован с Django
2. Статические файлы должны быть в `purchases/static/`
3. Используйте `{% load static %}` в шаблонах
4. Для production-среды запустите: `python manage.py collectstatic`

### 💡 Кастомизация

Все цвета определены как CSS переменные в `base.html`:

```css
:root {
    --primary: #0066ff;
    --primary-light: #e6f0ff;
    --success: #00b896;
    --danger: #ff4757;
    --warning: #ffa502;
    --bg-light: #f7f9fc;
    --border-color: #e0e7ff;
}
```

Измените эти значения для быстрого изменения всей цветовой схемы.
