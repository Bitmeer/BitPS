## 🔄 Процесс входа и выхода

### Вход:
1. Пользователь переходит на `/login/`
2. Вводит имя пользователя и пароль
3. Django проверяет учетные данные
4. При успешной аутентификации перенаправляет на `/` (request_list)

### Выход:
1. Пользователь кликает кнопку "Выход" в navbar
2. Сессия очищается
3. Пользователь перенаправляется на `/login/`

## ⚙️ Конфигурация

В `config/settings.py` добавлены:

```python
# Аутентификация
LOGIN_URL = 'login'                    # URL для входа
LOGIN_REDIRECT_URL = 'request_list'    # После входа → список заявок
LOGOUT_REDIRECT_URL = 'login'          # После выхода → страница входа
```

## 🔐 Безопасность

### Защищенные URL:
- `/` (request_list) - требует аутентификации
- `/create/` (request_create) - требует аутентификации
- `/login/` - доступна всем
- `/logout/` - требует аутентификации

### CSRF защита:
- `{% csrf_token %}` на странице входа
- `LoginRequiredMixin` на всех защищенных views

## 📝 Использование

### 1. Запустить сервер:
```bash
python manage.py runserver
```

### 2. Перейти на главную страницу:
```
http://localhost:8000/
```

### 3. Система автоматически перенаправит на:
```
http://localhost:8000/login/
```

### 4. Использовать тестовые учетные данные:
- **Username**: `testuser`
- **Password**: `testpass123`

## 🎯 Тестирование

### Тест входа:
1. Откройте http://localhost:8000/login/
2. Введите тестовые учетные данные
3. Нажмите "Войти в систему"
4. Вы должны попасть на страницу со списком заявок

### Тест выхода:
1. На странице со списком заявок кликните "Выход" в navbar
2. Вы должны попасть обратно на страницу входа
3. Попытка доступа к `/` перенаправит на `/login/`

## 🛠️ Кастомизация

### Изменить URL входа:
В `urls.py`:
```python
path('login/', views.CustomLoginView.as_view(), name='login'),
```

### Изменить перенаправление после входа:
В `settings.py`:
```python
LOGIN_REDIRECT_URL = 'request_list'  # Измените на нужный URL
```

### Изменить страницу входа:
В `views.py`:
```python
class CustomLoginView(LoginView):
    template_name = 'purchases/login.html'  # Путь к вашему шаблону
```
