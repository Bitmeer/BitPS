## 🔧 Кастомизация

### Изменить цвета:
Отредактируйте CSS переменные в `base.html`:

```css
:root {
    --primary: #0066ff;        /* Основной цвет */
    --success: #00b896;        /* Цвет успеха */
    --danger: #ff4757;         /* Цвет ошибки */
    --warning: #ffa502;        /* Цвет предупреждения */
    --bg-light: #f7f9fc;       /* Фоновый цвет */
    --border-color: #e0e7ff;   /* Цвет границ */
}
```

### Изменить шрифт:
В `base.html` используется Google Font "Inter". Измените на любой другой шрифт, отредактировав:

```html
<link href="https://fonts.googleapis.com/css2?family=YOUR_FONT:wght@300;400;500;600;700&display=swap" rel="stylesheet">
```
